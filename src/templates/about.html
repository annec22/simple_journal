About Simple Journal

This is a simple web app project which is still in-progress.

Future improvements include:

  • Functions: Edit, delete and search for entries, Import entries from both local and Google drive
  • Design: Improving form input for adding and editing entries
  • Quality: Form validation

Technologies used to build this application are Python, Flask, MongoDB, Bootstrap and a little bit of Google API.
